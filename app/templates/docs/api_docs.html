Webhooks & HMAC Signatures

When a payment completes, fails, or requires manual verification (UNKNOWN), we send a POST request to your target_callback_url.

Headers
  • X-VS-Signature: HMAC SHA256 signature
  • X-VS-Timestamp: Unix timestamp of the request
  • X-VS-Transaction: Vitta Setu Transaction ID
Verifying Signatures (Python Example)
import hmac, hashlib, json

def verify_webhook(secret, payload_dict, timestamp, received_signature):
    # payload_dict must be serialized with no spaces between separators
    message = f"{timestamp}.{json.dumps(payload_dict, separators=(',', ':'))}"
    expected_signature = hmac.new(
        secret.encode('utf-8'),
        message.encode('utf-8'),
        hashlib.sha256
    ).hexdigest()
    
    return hmac.compare_digest(expected_signature, received_signature)
